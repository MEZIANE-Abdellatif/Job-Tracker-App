# JOB TRACKER — frontend

Next.js client for **JOB TRACKER**. Full setup, environment variables, and architecture notes live in the **[repository root README](../README.md)**.

## Quick start

```bash
npm install
# Create .env with NEXT_PUBLIC_API_BASE_URL pointing at your API (e.g. http://localhost:3001)
npm run dev
```

App runs at **http://localhost:3000** (see `package.json` script).

## Scripts

- `npm run dev` — development server  
- `npm run build` — production build  
- `npm run start` — serve production build  
- `npm run lint` — ESLint
